# GoBid

A Go service project demonstrating Docker Compose deployment and Go backend logic.

---

## 🔧 Prerequisites

- Go 1.20+ installed
- Docker & Docker Compose installed

---

## 🚀 Quick start (local)

```bash
cd go/05-gobid
go run main.go
```

---

## 🚀 Quick start (Docker)

```bash
cd go/05-gobid
docker compose up --build
```
